## What was built
A unified Quarto book from 25+ scattered JASP development markdown files, user-guide docs, Mattermost posts, and jasp-agent-instructions. Goal: single coherent guide for humans wanting to write JASP modules. Audience: both internal and external developers.

## Key patterns from real modules (for future reference)
- **TTestIndependentSamples**: main function delegates to output functions, each checks `!is.null(jaspResults[["key"]])` for caching, tables use `dependOn()`, `showSpecifiedColumnsOnly`, `position`
- **Correlation**: `.corrMainResults()` checks both `jaspResults[['mainTable']]` and `jaspResults[['results']]`, returns `$object` if cached
- **Containers**: `createJaspContainer()` with `dependOn()` and `position`, used in Descriptives for grouping related outputs
